
# React + NodeJS Chat App

## Cách chạy:

### 1. Cài backend
```bash
cd server
npm install
node server.js
```

### 2. Cài frontend
```bash
cd client
npm install
npm start
```

### 3. Import file SQL
- Mở MySQL và import file `db.sql` để tạo bảng và user mẫu.
